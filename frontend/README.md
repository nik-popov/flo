# Flo frontend

This package contains the Flo fulfillment React single-page app powered by Vite.

## Prerequisites

- Node.js 20+
- npm 10+

Install dependencies:

```bash
npm install
```

## Development

Start the local dev server (with hot reload):

```bash
npm run start
```

## Testing

Run the Vitest suite:

```bash
npm test
```

## Production build

Create an optimized bundle in `dist/`:

```bash
npm run build
```
